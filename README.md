# 🩺 MediGuardian AI

An AI-powered healthcare assistant that analyzes medical reports and provides personalized health insights, recommendations, and wellness guidance.

## 📌 Project Overview

MediGuardian AI helps users understand their medical reports by using Artificial Intelligence to generate easy-to-understand health summaries and recommendations.

The application allows users to upload medical reports and receive:

- 📊 Health Score
- 🩺 Report Summary
- 🍎 Diet Recommendations
- 💪 Exercise Suggestions
- 🌿 Lifestyle Tips
- 💧 Water Intake Recommendations
- 😴 Sleep Recommendations
- 📅 Follow-up Guidance
- 📄 Downloadable AI Report

## 🚀 How to Run

### Clone the Repository

```bash
git clone https://github.com/yourusername/MediGuardian-AI.git
```

### Go to Project Folder

```bash
cd MediGuardian-AI
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

Open:

```
http://127.0.0.1:5000
```

## 🎯 Problem Statement

Medical reports often contain complex medical terminology that is difficult for patients to understand.

MediGuardian AI simplifies medical reports using AI and provides personalized health recommendations, enabling users to better understand their health condition and take preventive actions.

---

## 📈 Workflow

1. User Login
2. Upload Medical Report
3. AI Processes Report
4. Extract Medical Information
5. Generate Health Score
6. Display Dashboard
7. Provide Personalized Recommendations
8. Download AI Report
